#测试题

    团队最近爱上了一家自助烧烤店，在这家店你可以吧台挑选任意的蔬菜，肉和酱等来作为你的餐食。当你挑好材料后，就拿到烧烤架上烧煮。

    你的任务是估量平均每位顾客的餐食准备时间。假定每位顾客都会去挑选随机的5份材料，从每个吧台取材料的时间为10s，共有20个吧台每个吧台同时只能有1个人使用，烧烤架烤一份食物要3分钟，但能同时烤8份食物。当吧台和烧烤架满的时候，顾客只能排队等待。

    估算以下场景的平均时间：

    只有1个顾客时
    同时有8个顾客
    同时有25个顾客
    有25个顾客，但两两相距1分钟
    同时100个顾客
    同时100个顾客，但两两相距2分钟

    加分项：

    估算顾客在烧烤架的平均等待时间
    估算顾客在吧台的平均等待时间
